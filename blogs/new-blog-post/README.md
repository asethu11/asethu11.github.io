# New Blog Post

## Required Files

### Thumbnails
- `thumbnail.png` - Main thumbnail image (recommended: 1200x630px)
- `thumbnail.jpg` - Alternative thumbnail format

### Images Folder
Place your blog images in the `img/` folder:
- `img/slide1.jpg` - First image
- `img/slide2.gif` - Second image (can be GIF or static)
- `img/slide3.gif` - Third image (can be GIF or static)

## Customization

1. Rename the folder from `new-blog-post` to your desired blog name (use kebab-case)
2. Rename `new-blog-post.html` to match your folder name
3. Update the HTML content with your blog post
4. Add your thumbnail images
5. Add your blog images to the `img/` folder
6. Update `index.html` with the correct path and title
