The brief

Point One Zero produces output across 8 distinct formats before the system existed. Each was styled from scratch — brand color applied wherever it felt right, type sizes eyeballed, spacing drifted between surfaces. A pull quote set at 40px in a white paper had no relationship to the 40px heading on the website. There was no shared definition of what the firm looked like.

The goal was one source of truth that would govern everything — without constraining it. A system that could underpin a pitch deck and a social carousel and a stakeholder report, all with the same visual logic underneath.

Key decisions

Token-first, light-first. Every color, type size, spacing value, radius, and shadow is a CSS custom property on :root. Everything downstream inherits. Light is the default — dark is an opt-in scope for surfaces that need it (pitch decks, product chrome). Never via prefers-color-scheme.

The surface ladder. Hierarchy on light is hard — white cards on a white canvas look flat. The solution is a four-step surface ladder that steps down into faint gray (--canvas → --surface-1 → --surface-2 → --surface-3), supplemented by a soft shadow. A featured card settles one step into the page instead of changing color.

"

The brand is a noun, not a color. It appears where it carries meaning and nowhere else.

PZ Point One Zero · Design Language, v0.5.0

Brand restraint. #009FEF is assigned exactly 8 named contexts: the wordmark dot, one hero word, focus rings, link underlines on hover, one featured metric per page, active toggle states, editorial ordinal markers, and pull-quote glyphs. If it's not on the list, it doesn't appear. The discipline of that constraint is what makes the blue land when it does.

The constraint
0

Hardcoded hex values in component markup across all 12 kits. Every color is a CSS variable — a single line edit propagates everywhere.

The kits

12 output formats, all extending the same foundation. Web pages, letter-size documents, 1920×1080 slide decks, 1080×1350 social carousels, and 1080×1440 print white papers. Each kit ships its own spec, component vocabulary, copywriting budgets, common mistakes, and an agent prompt — the system was designed for AI-assisted authoring from the start.

Typography

One face, eleven steps

Inter Variable from 80px down to 12px. Hierarchy is size and tracking — not family or color. Display uses weight 600 with aggressive negative tracking. The eyebrow inverts with positive tracking (+0.4px) to read as taxonomy, not statement.
